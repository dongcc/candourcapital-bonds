# Candour Capital Bonds Output Format

## Full List Output

When listing all bonds (e.g., via `fetch_candour_bonds.py`), present:

- Header: “Candour Capital Bonds — total: N (AUD: X, USD: Y)”
- Group by currency: **AUD** and **USD** sections
- Each bond line:
  ```
  {issuer} ({currency}) — {coupon}% — {issue_date} to {maturity_date} — Rating: {rating}
  ```
  - If `maturity_date` is `Perpetual`, show “to Perpetual”
  - Rating is shown as provided (e.g., `Aa2/AA-`)

Example:
```
Commonwealth Bank (AUD) — 6.4% — 5-Mar-2026 to 5-Mar-2046 — Rating: Aa2/AA-
UBS (AUD) — 7.125% — 13-Feb-2026 to Perpetual — Rating: A2/A-
```

## Dashboard Output (HTML)

Generated by `generate_dashboard_unified.py` with `--currency ALL`, `AUD`, or `USD`.

Features:
- **Ratings distribution**: Bar chart showing count of bonds per rating (A/B/C classes).
- **Maturity schedule**: Horizontal bar chart, each bar spans from Issue year to Maturity year (or 2100 for Perpetual).
  - Y-axis: Issuer (shortened to 13 chars + “…”) plus coupon (e.g., “Commonwealth Bank (6.4%)”).
  - X-axis: Year scale, min = earliest issue year in the set, max = 2100.
  - Bar colors by rating class:
    - A class (Aaa, Aa, A) → green shades (darker = higher grade)
    - B class (Baa, Ba, B) → yellow shades (darker = higher grade)
    - C class (Caa, Ca, C) → red shades (darker = higher grade)
  - Perpetual bonds: bar ends at 2100 and the word “Perpetual” is displayed inside the chart area next to the bar tip.
- Filter buttons (for “All” view) to show/hide AUD or USD.
- Responsive dark theme, sortable table with Term Sheet and Apply links (if present in data).

File naming:
- `all_bonds_<YYYYMMDD>_<HHMMSS>.html`
- `aud_bonds_<YYYYMMDD>_<HHMMSS>.html`
- `usd_bonds_<YYYYMMDD>_<HHMMSS>.html`

## Notes

- Generated timestamp is embedded in the HTML.
- Serve via an HTTP server on port 8002 from the workspace root to access all skill dashboards under `/skills/<skill-name>/...`.
